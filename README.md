# 🩺 AI-Based Skin Disease Detection with Visual Highlight

This project is an AI-powered skin disease detection system developed using Python and Machine Learning.

It predicts skin diseases from images and highlights the affected region using image processing techniques.

---

## Features

- Upload a skin image
- Automatic disease prediction
- Visual highlight of affected area
- Image preprocessing using OpenCV
- HOG feature extraction
- Machine Learning classification
- Result image generation

---

## Technologies Used

- Python
- OpenCV
- NumPy
- Scikit-image
- Scikit-learn
- Joblib
- Tkinter
